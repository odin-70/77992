Bienvenidos a la Clase 4

Vinculos

Para editar el aspecto visual de nuestros vinculos debemos tener en cuenta que los mismos estan compuestos por 4 estados:

  • Reposo(a: link)
  • Visitado(a: visited)
  • 'Hover'(a: hover)
  • Activado(a: active)

Debemos respetar el orden de las pseudo-clases cuando escribamos nuestro CSS. No puede faltar ninguna.

Flexbox

un metodo de CSS para ordenar elementos en filas o columnas. Ademas sirve para que los elementos sean "flexibles" dentro de un contenedor.
